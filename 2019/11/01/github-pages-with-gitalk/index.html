Hexo

遇到的问题

  1. Error: Not Found.

没有填对repo等对应的信息

  1. Error: Validation Failed

location.href要改成location.pathname

  • 或者可以改成md5，这样就不超过50个字符啦。可以参考Hexo里的代码, 在themes/next/scripts/helpers/engine.js文件里面

用Hexo来搭建github-pages博客

  1. 安装hexo npm i -g hexo-cli

  2. hexo init gh-pages & cd gh-pages

  3. 安装 next 主题

    git clone https://github.com/theme-next/hexo-theme-next themes/next

  4. 切换主题, 在主目录下的 _config.yml 看到theme: landscape 修改成theme: next

  1. 配置Gitalk, 在 next 文件夹下的_config.yml

    1. 如果没有github app，点这里注册一个, 申请方法参考here

    2. 修改文件 /theme/next/layout/_partial/comments.swig

  2. hexo clean & hexo deploy

  3. git init时要先删除themes/next里的.git文件夹
